# 运算符
## 优先级
#### the first

| 符号 | 作用 | 描述 |
| :----: | :----: | :----: |
|  *  | 算术运算 | `相乘` |
|  /  | 算术运算 | `相除` |
|  %  | 算术运算 | `求余` |
|  && | 逻辑运算 | `AND` |
|  & | 位数运算 | 双目运算符 |
| << | 位数运算 | 双目运算符 |
| >> | 位数运算 | 双目运算符 |
| &^ | 位数运算 | 双目运算符 |

| 符号 | 作用 | 描述 |
| :----: | :----: | :----: |

| != | 条件判断 | `不等于` |
| <  | 条件判断 | `小于` |
| <= | 条件判断 | `小于或等于` |
| >  | 条件判断 | `大于` |
| >= | 条件判断 | `大于或等于` |
| <- | channel发送&接受 | `channel &&` |
